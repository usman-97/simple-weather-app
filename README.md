# Simple Weather App

Simple Weather App is a react app which fetches data from [simple-weather-api](https://github.com/usman-97/simple-weather-api) and then show it to user. Check [Simple Weather App](https://simple-weather-b3rj2p5gv-usman-97.vercel.app/)!
**NOTE: Simple Weather API has been deployed to Render with free plan where free instance will spin down with inactivity, which can delay requests by 50 seconds or more.**

## Features

- Search city by name to check it's weather
- Real time search
